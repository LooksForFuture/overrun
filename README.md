# overrun
2D top down shooter game made in C as an educational project using custom ECS.

## project structure
The project is divided into re-usable modules. The engine module is a small game development toolkit which doesn't containg game specific code. The game directory contains the game specific code and links to the engine module. And the submodules directory contains submodules which are thid-party libraries.

The purpose and the philosophy behind each part, is described in its markdown files and not in the comments. The comments are for describing the axioms, rules, side notes, exceptional situations, etc.
